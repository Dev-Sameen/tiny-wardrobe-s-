# 👗 Tiny Wardrobe

A modern kids clothing eCommerce website built using **HTML, CSS, and Vanilla JavaScript**.

## ✨ Features

* 🛍️ Product listing with sizes & colors
* 🛒 Add to cart system
* ➕ Quantity management
* 📦 Order form with validation
* 🎉 Thank you screen after order
* 🔔 Toast notifications
* 📱 Fully responsive design
* ⚡ Smooth animations

## 💰 Pricing

All prices are displayed in **PKR (Pakistani Rupees)**.
